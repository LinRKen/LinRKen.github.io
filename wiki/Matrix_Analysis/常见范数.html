常见范数

向量范数

向量范数
映射 $\lVert \cdot \rVert: \mathbb{C}^{n} \rightarrow \mathbb{R}$, 若满足:

  1. 正定性: $\lVert x \rVert \geq 0$, 且 $\lVert x \rVert = 0 \Leftrightarrow x = 0$;
  2. 齐次性: $\lVert \alpha x \rVert = |\alpha| \lVert x \rVert$;
  3. 三角不等式: $\lVert x + y \rVert \leq \lVert x \rVert + \lVert y \rVert$.
    则称 $\lVert \cdot \rVert$ 是 $\mathbb{C}^{n}$ 向量范数.

向量范数具有以下性质:

  1. $\lVert 0 \rVert = 0$;
  2. $x \neq 0$ 时, $\left\lVert \frac{1}{\lVert x \rVert} x \right\rVert = 1$;
  3. 对任意 $x \in \mathbb{C}^{n}$, $\lVert -x \rVert = \lVert x \rVert$;
  4. 对任意 $x, y \in \mathbb{C}^{n}$, $\lvert \lVert x-y \rVert \rvert \leq \lVert x-y \rVert$.

常见向量范数

  1. $p$ 范数: $\lVert x \rVert_{p} = \left( \sum_{i=1}^{n} |x_{i}|^{p} \right)^{1/p}$, $p \geq 1$;
  2. $1$ 范数: $\lVert x \rVert_{1} = \sum_{i=1}^{n} |x_{i}|$;
  3. $2$ 范数: $\lVert x \rVert_{2} = \sqrt{\sum_{i=1}^{n} |x_{i}|^{2}}$;
  4. $\infty$ 范数: $\lVert x \rVert_{\infty} = \max_{1 \leq i \leq n} |x_{i}|$.

矩阵范数

矩阵范数
设 $A = (a_{ij}) \in P^{m \times n}$, 映射 $\lVert \cdot \rVert: P^{m \times n} \rightarrow \mathbb{R}$, 若满足:

  1. 正定性: $\lVert A \rVert \geq 0$, 且 $\lVert A \rVert = 0 \Leftrightarrow A = 0$;
  2. 齐次性: $\lVert \alpha A \rVert = |\alpha| \lVert A \rVert$;
  3. 三角不等式: $\lVert A + B \rVert \leq \lVert A \rVert + \lVert B \rVert$.
    则称 $\lVert \cdot \rVert$ 是 $P^{m \times n}$ 向量范数.

常见矩阵范数

  1. $1$ 范数: $\lVert A \rVert_{1} = \max_{1 \leq j \leq n} \sum_{i=1}^{m} |a_{ij}|$;
  2. $2$ 范数 (又称为 Frobenius 范数 $\lVert A \rVert_{F}$): $\lVert A \rVert_{2} = \sqrt{\rho(A^{*}A)}$;
  3. $\infty$ 范数: $\lVert A \rVert_{\infty} = \max_{1 \leq i \leq m} \sum_{j=1}^{n} |a_{ij}|$.

算子范数的性质

  1. 算子范数是所有与向量范数 $\lVert x \rVert_{a}$ 相容的矩阵范数中最小的.
  2. 算子范数是自相容矩阵范数.

常见算子范数

  1. $1$ 范数 (极大列和): $\lVert A \rVert_{1} = \max_{1 \leq j \leq n} \sum_{i=1}^{m} |a_{ij}|$
  2. $\infty$ 范数 (极大行和): $\lVert A \rVert_{\infty} = \max_{1 \leq i \leq m} \sum_{j=1}^{n} |a_{ij}|$
  3. $2$ 范数 (谱范数): $\lVert A \rVert_{2} = \sqrt{\rho(A^{*}A)}$

谱范数的性质

  1. $\lVert A \rVert_{2} = \lVert A^{H} \rVert_{2} = \lVert A^{T} \rVert_{2} = \lVert \bar{A} \rVert_{2}$
  2. $\lVert A^{H}A \rVert_{2} = \lVert AA^{H} \rVert_{2} = \lVert A \rVert_{2}^{2}$
  3. 酉不变性: $\lVert A \rVert_{2} = \lVert UA \rVert_{2} = \lVert AV \rVert_{2} = \lVert UAV \rVert_{2}$

  5. $\lVert A \rVert_{2}^{2} \leq \lVert A \rVert_{2} \cdot \lVert A \rVert_{\infty}$
